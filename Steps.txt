Adding Typescript to Project

In cmd terminal:

1) npm install typescript --save-dev

2) TypeScript (.ts) must be compiled to JavaScript (.js) before the browser can use it.
    in cmd: npx tsc

3) In Razor page (Index.cshtml):
   	add it in path: wwwroot/js/client-index.js
	add this small script above
 	@section Scripts {
    		<script src="~/js/client-index.js"></script>
	}
